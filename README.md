# PaccY

Version: 1.0.0

PaccY is een vrolijke Pac-Man-achtige arcadegame in plain HTML, CSS en JavaScript. De game heeft geen dependencies, geen build-stap en geen externe assets.

## Starten

Open `index.html` in een moderne browser.

## Features

- Meerdere handgemaakte levels
- Cursorbesturing met vloeiende tile movement
- Vier ghosts met eigen gedrag
- Pellets, power pellets, fruitbonussen en highscore
- Muziek en geluidseffecten via de Web Audio API
- Responsive canvas-layout
- Geen npm, frameworks, downloads of build tools

## Besturing

- Pijltjestoetsen: bewegen
- Spatie: pauze
- R: herstart
- M: geluid aan/uit

Audio start na de eerste klik of toetsdruk, omdat browsers dat voor geluid vereisen.
